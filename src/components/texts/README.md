## Instalação

Para usar este arquivo CSS em seu projeto, você não precisa instalar nada. Basta copiar o arquivo `texts.css` para a pasta de seu projeto e importá-lo em seus arquivos JavaScript.

## Uso

Para aplicar as regras CSS globais definidas no `texts.css`, você precisa importar o arquivo CSS no arquivo JavaScript principal, como mostrado abaixo:

```javascript
import './texts.css';
```

Depois de importar o arquivo CSS, as regras CSS serão aplicadas a todos os elementos HTML na sua página.

## Regras CSS

Este arquivo CSS define as seguintes regras CSS globais:

- A fonte padrão é "Inter".
- O espaçamento entre as linhas padrão é definido por cada seletor específico.
- O espaçamento entre letras padrão é definido por cada seletor específico.
- O alinhamento padrão é à esquerda.
- Verifique os elementos referidos e suas classes para uso devido

Você pode personalizar as regras CSS globais adicionando ou modificando as regras no arquivo `texts.css`.
